# 第130-148话 — 罗天大醮后篇 / 全性攻山篇

> 来源：130-148.pdf（273页），逐页阅读提取

## 主线剧情

### 罗天大醮善后（130-133话左右）
- **罗天大醮结束**，张楚岚夺冠但拒绝继承天师之位。各方势力重新洗牌。
- **风正豪**（风家家主，银发老者）在龙虎山后与女儿通话。提到"这次龙虎山已经没咱们风家什么事了"，让沙燕下山休息。风家在此事件中偏旁观者。
- 一个戴眼镜的黑发男子（疑似哪都通高层）在高楼中说"这个暂时还不能告诉你们……总之大家要清楚，我们所做的努力不会白费就好！"——暗示哪都通在罗天大醮背后有自己的布局。
- **王也**（蓝发丸子头）与楚岚交谈："楚岚做出了自己的选择……而来龙虎山出头应该也是你自己的选择吧……既然选择已经做出来了再纠结也没用。"安慰楚岚不要自责。王也说"我这人不爱被卷进麻烦，但你有麻烦的话我一定帮你"——体现王也的性格和对楚岚的友谊。

### 全性攻山 / 龙虎山之战（134-140话左右）
- **全性攻龙虎山天师府**。各路老前辈聚集讨论对策。
- 一位老者（天师府前辈）愤怒说"给我狠狠的修理张怀义那个大耳朵的孙子！"——暗指张楚岚与张怀义（三十六贼之一）的关系引发天师府内部矛盾。
- **张灵玉**（蓝发道士）与一位师叔对战到极限："不过我已经到极限了……""师叔！要不要让晚辈一招？""可以，你先出手吧……"——张灵玉在全性攻山中积极守卫天师府，展示五雷正法。
- **张灵玉使出五雷正法**（"雷正法——为五雷，实为五炁"），蓝色雷电笼罩全身——这是天师府核心功法，"人一降生，先天之炁存于体内，后纳之炁可分五行"。
- 战斗中张灵玉被打得很惨，有人喊"师弟，你……该！""张灵玉！你还能继续下去么？"——但他坚持战斗。
- 战斗场面有人被打飞，围观群众震惊（"……"满屏省略号），**冯宝宝**（长发女子）也在场观战。

### 张楚岚的秘密曝光（136-139话左右）
- **老一辈聚会讨论张楚岚**。一个黑板/白板上写着各势力年轻一辈名字：**诸葛青、风燕、王也、张楚岚**——被圈为重点关注对象。
- 提到"还有老陆也是，一个个一把年纪了，唯恐天下不乱"——指**陆家**（陆瑾、陆琳的家族）也在搅局。
- "老的都还没消停……现在小的又开始崭露头角了么……""说到底，根根还是在于三十六贼……"——将矛盾追溯到**甲申之乱**和**三十六贼**。
- 提到"张怀义啊张怀义，你说你们为什么偏偏和那个人搞在一起"——暗示张怀义与无根生的关系是所有矛盾的源头。

### 楚岚拒绝天师位后的反应（140-143话左右）
- **张楚岚拒绝天师位**让在场老前辈震惊："这个张楚岚可太能搞怪了！""不要？"
- **风家白发女子**（风星潼/沙燕？）在山下打电话："老爹你们也下山了？好，我等着你们……"——风家离开龙虎山。
- 冯宝宝来到一处殿堂前（天师府建筑），独自进入。
- 随后有人被**捏碎了手**（一只手被狠狠攥碎的画面），发出惨叫"哇哇！我的手！"——疑似全性成员的袭击。

### 全性撤退与追击（144-148话左右）
- "别跑！"——有人追击逃跑的全性成员。**冯宝宝**戴着大草帽持刀追杀，使出"嗨嗨嗨嗨"的连续攻击——宝宝的标志性战斗风格。
- "就不能放了我们么？"——被追的全性成员求饶。
- "陆前辈，恕见识广，晚辈是参不透……""这老者用的究竟是什么功夫！"——**陆家前辈**（戴蓝帽的壮汉）使用了让人看不懂的功夫，疑为**拘灵遣将**。
- 战斗场面：蓝色能量鱼形攻击、蓝色符咒防护罩——"这是！我们天师府的五雷符咒！？""嘿，这些护身法宝都是炼器师经年累月的心血所成……所以对付起来也没什么取巧的办法……只能拿出实打实的威力来……"
- 最后一位金发壮汉（疑似全性高手）提到"到损伤时血肉也不是寻常的再生方法，而是连炁都能修补……""据说要是能炼到第三重，完美的炁化……那人也就离羽化飞升不远了！"——暗示**逆生三重**功法（马仙洪的能力），炼到极致可以羽化飞升。

## 关键设定补充

- **五雷正法**：天师府核心功法，以先天之炁为基础，分化五行之雷。张灵玉在战斗中展示。
- **逆生三重**：炼到第三重可实现完美炁化，修补炁本身，接近"羽化飞升"——这是马仙洪的核心能力路线。
- **三十六贼**：甲申之乱的核心人物团体，张怀义是其中之一，是当前所有矛盾的历史根源。
- **天师位继承**：张楚岚在罗天大醮夺冠后有资格继承天师位，但拒绝了。老前辈们讨论"头名的张楚岚不要……那就传给第二位的吧！"

## 篇章意义

这段是罗天大醮和全性攻山的过渡/高潮，核心事件：
1. 张楚岚拒绝天师位 → 打破各方预期
2. 全性攻龙虎山 → 天师府面临危机
3. 张灵玉的成长 → 五雷正法首次完整展示
4. 各方势力重新评估年轻一代 → 为后续篇章铺垫
5. 三十六贼/甲申之乱被反复提及 → 伏笔加深
