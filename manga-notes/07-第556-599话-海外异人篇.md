# 第556-599话 — 海外异人篇 / 纳森卫事件

## 主线剧情

### 巴伦与异人小队（556-562话左右）
- **巴伦**（金发外国人）带领一支小队在野外/山林中活动。有人感到身体异变："消失了，那让人不安的感觉……那一是我的错觉么……那种沁入身体的感觉，丝滑啊""空气好像都变了……好像有微量的……""没错……每次呼吸之后身体都感觉变轻了一样……"
- 巴伦身旁有一个年轻人问"巴伦先生在干么？"——巴伦似乎在做某种仪式或者感知训练，刚才失去的体力在恢复。
- 巴伦与人合作："没问题！哈哈！红眉毛，看以后咱们还会有很多合作的机会"，"很好！巴伦！需要什么尽管说就好"——拿到一个免费的礼物/法器，白胡子老者递给他物品。
- 小队在山林中遭遇激烈战斗和追逐。

### 甲申之乱回忆 / 能力觉醒（560-568话左右）
- 出现**陈朵**（粉红发女孩）相关剧情。有人在雨中追击她。一个蓝脸/灰脸的异人出现在门口——"这不是我的混合体吗！你……这里是……"——涉及**神机百炼**的设定。
- 战斗场面：冰/水属性攻击，"这种水平……比起和我过上的时候……退步了""之前在那个山谷里也是……"——有人在评价对手的实力变化。
- 一个满身伤痕的光头壮汉（疑似全性高手或海外异人），"原来如此……他废弃的限制就是千遍来死……那现在碧游村的友人面前增加对手的实力……""不管这么会合之后有多大的风险……一切只是为了去死对方的弱点……"——在分析某种能力的机制。

### 陈朵线 / 神机百炼（568-575话左右）
- **陈朵**（粉色长发女孩）浑身覆盖红色液态物质（血？炁？），在废墟/山地中战斗。被巨型怪物攻击。
- 红色触手/丝线从陈朵体内延伸，她操控这些进行攻击和防御——这是**神机百炼·体呈器**的表现。
- 一段关于"珍珠"的对话："珍珠五岁左右开始就会经常在名有角落找到……""太爷和珍珠的父亲也是精挑细选的不行""太爷和珍珠的父亲对……""珍珠本人也是跟相中了两个？哪两个？""只是太爷的态度已经很明确了"——暗示陈朵（小名珍珠？）的家族背景和联姻安排。
- 陈朵满身鲜血仍在战斗，被打倒后说"嗯嗯嗯嗯一大堆……自杀啊嘶……""谁要自杀？"——性格倔强不认输。

### 海外势力 / 贝斯迪亚（575-585话左右）
- 出现海外异人势力。一个戴面具的组织在丛林中行动，使用军事化装备（直升机、自动武器、防弹衣）。
- "捕获目标 3-0773 3-932""捕获目标 3-0094"——他们在系统性地捕获异人/灵虫（"灵虫——窝媳"）。
- 紫发男子和金发壮汉交谈——"跟我去乐园坐坐，至少恢复一下体力"。紫发男递给壮汉一颗红色果实/丹药。
- **纳森卫**（金发西装男，贝斯迪亚组织的"王"）登场："王！您受着伤应该加紧纳森卫？""请接受我们的帮助，今后我会尽力辅佐您""见到您太激动了！原谅我！还未曾自我介绍！""罗恩，看着吧，也许我会创造奇迹，也许我通过这判就能……"——纳森卫是一个野心勃勃的海外异人领袖。
- **贝斯迪亚**——海外异人组织名。他们的目的和中国异人界有关："后来大量海外之人登陆，成为了大陆的新主人""你们时还传播着祖先的力量感觉关系""而你的控制权，他也从把转到了他们的手上"——暗示海外势力与中国异人的历史渊源。

### 异空间 / 王也困局（580-590话左右）
- **王也**被困在一个宇宙般的异空间中（星空背景、浮石平台）。白胡子老者也在其中。
- "这里是无限的吗？""感官上已经在这里持续行进了一个小时以上……""如果真的是这样，两者结合已经解决了未来的事了""或许……这里的时间和外面不一样""除了自己以外也无感受不到任何气息……温度也没有以上变化……连炁压也没有变化""这里是某个人造的空间"——王也在分析异空间的规则。
- "华金师！""半会到那个混蛋，魔在……要我！"——有人呼喊华金师，还有怪物在这个异空间中。
- "华金师……怎么样了……""那个怪物……还在坚持吗？""已经看过多次了！""没必要去理它到""15年，那个东西有就没有改变""一步一步的向我们步步逼近"——一个追了他们15年的怪物。

### 纳森卫 vs 中方势力（590-599话左右）
- 纳森卫的人与中方势力发生冲突。战场上有现代武器（枪械、装甲车）与异人术法的混战。
- "厉害！""纳森卫！""太厉害了！""我们是无敌的！""阻止他！"——纳森卫在战场上表现强势。
- 战斗中出现大范围冲击波、紫色能量球，多人被震飞。
- "了解的很清楚啊，从'蚁巢'继承来么？""你……真的是贝斯迪亚？""好了，我不管为什么你外形和资料不符……""如果你真的是贝斯迪亚就不能放走你！""我们不会伤害你！还他给你带来的！"——中方试图确认贝斯迪亚的身份。
- "福佑宝！这个生灵，也算真是他的御定"——某人对贝斯迪亚/纳森卫的能力表示认可。
- 一位灰袍人对外国老者和女人说"我给你们最后的机会"，回答："滚。"——然后直升机飞走，灰袍人独立在彩虹色的地面上。极其霸气的画面。
- "城炉大人！""那就来吧"——出现一个被称为"城炉"的角色，全身红色肌肉暴露（类似逆生状态），是全性或海外势力的高手。
- 最后的战斗延续到城镇中——"真快……"——高速战斗，城市建筑被破坏。

## 关键设定

- **神机百炼·体呈器**：陈朵的能力，可以让身体化为武器/触手，红色液态物操控——暗示和甲申之乱有关。
- **贝斯迪亚 / 纳森卫**：海外异人组织，有军事化运作（编号系统捕获灵虫/异人），"王"纳森卫是领袖。
- **异空间**：王也被困的人造空间，时间流速不同，内有追了15年的怪物，华金师等人也被困其中。
- **"蚁巢"**：海外异人势力的某个分支/前身，和贝斯迪亚有继承关系。
- **海外异人历史**：暗示海外异人曾"登陆大陆"成为"新主人"，与中国异人的祖先力量有传承关系。

## 篇章意义

556-599话是**海外异人线**的核心展开：
1. 世界观从国内扩展到全球——海外势力（贝斯迪亚/纳森卫）正式登场
2. 陈朵的神机百炼能力深入展示，家族背景揭示
3. 王也被困异空间，引出新的谜团（15年怪物、华金师）
4. 巴伦作为中外异人的桥梁角色，连接多条线索
5. 现代军事力量（枪械/直升机）与传统异人术法的碰撞
