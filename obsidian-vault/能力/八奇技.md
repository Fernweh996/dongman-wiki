# 八奇技

> 异人界最强的八种绝技，每一种都拥有改变战局的力量。

## 概述
八奇技是异人界中最顶级的八种能力，每一种都极为强大且稀有。持有者往往是异人界的核心人物。

## 已知八奇技
| 奇技名 | 持有者 | 描述 |
|------|------|------|
| 双全手 | [[张楚岚]] | 左手画圆右手画方，同时施展两种不同技能 |
| 风后奇门 | [[王也]] | 空间操控，可扭曲、切割、连接空间 |
| 他化自在天魔咒 | [[十七]] | 具体效果神秘，持有者隐居[[纳森岛]] |
| 神机百炼·体呈器 | [[陈朵]] | 将自身化为器具，红色丝线/触手战斗 |
| 拘灵遣将 | [[陆瑾]]/[[风天养]]（风家）/王家 | 多个家族持有不同版本，完整版极为稀少 |
| 逆生三重 | [[马仙洪]] | 以自身生命力为代价的逆转能力 |
| 通天箓 | [[陆瑾]] | 陆家传承 |
| 炁体源流 | [[张楚岚]] | 增强炁的运用 |

## 特点
- 每种奇技都是异人界顶尖能力
- 持有者数量极少
- 部分奇技有不同版本（如拘灵遣将有完整版和残缺版）
- 一人可持有多种奇技（如[[张楚岚]]同时拥有双全手和炁体源流）
